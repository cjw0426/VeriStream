# VeriStream 复现指南

## 环境与数据

安装 PyTorch/torchvision 后执行：

```bash
conda create -n simplestream-qwen3 python=3.10 -y
conda activate simplestream-qwen3
pip install -r requirements-qwen3.txt
```

OVO-Bench 数据放置为：

```text
data/ovo_bench/ovo_bench_new.json
data/ovo_bench/chunked_videos/
```

模型权重、数据、结果和日志不进入 Git。8 卡机器运行 Accelerate 时固定设置：

```bash
export NCCL_P2P_DISABLE=1
export NCCL_IB_DISABLE=1
export OMP_NUM_THREADS=1
export MKL_NUM_THREADS=1
export TOKENIZERS_PARALLELISM=false
```

## 测试

```bash
python -m unittest discover -s tests -v
```

## Recent baseline

```bash
CUDA_VISIBLE_DEVICES=0,1,2,3 accelerate launch --num_processes=4 \
  main_experiments/eval_qwen3vl_ovo.py \
  --model_path /path/to/Qwen3-VL-8B-Instruct \
  --anno_path data/ovo_bench/ovo_bench_new.json \
  --chunked_dir data/ovo_bench/chunked_videos \
  --result_dir main_experiments/results/ovo_qwen3vl_recent4 \
  --frame_selection recent --recent_frames_only 4 \
  --chunk_duration 1.0 --fps 1.0 --max_qa_tokens 256
```

该入口也支持 `uniform`、`clip_topk`、`recent_clip_topk`、文本 memory 等对照设置。
例如 `Recent4 + CLIP-topk16` 使用：

```bash
--frame_selection recent_clip_topk \
--recent_frames_only 4 --supplemental_frames 16
```

## Hybrid-4 Forward

```bash
CUDA_VISIBLE_DEVICES=0,1,2,3 accelerate launch --num_processes=4 \
  main_experiments/eval_qwen3vl_ovo_dual_role_forward.py \
  --perception-model-path /path/to/Qwen3-VL-8B-Instruct \
  --reasoning-model-path /path/to/Qwen3-VL-8B-Instruct \
  --result-dir main_experiments/results/veristream_dual_role_hybrid4_forward \
  --chunk-duration 1.0 --fps 1.0 --coarse-stride 12 \
  --max-frames-per-observation 8 --max-actions 4 \
  --max-qa-tokens 256 --final-recent-frames 4
```

如需断点续跑，保持同一个 `--result-dir`。程序会读取 `rank_*/results_incremental.jsonl`，
跳过已经完成的样本。若遇到视频 EOF 或 FFmpeg 资源错误，可先设置：

```bash
export DECORD_EOF_RETRY_MAX=20480
```

## 证据索引与查询

单视频建立索引：

```bash
CUDA_VISIBLE_DEVICES=0 python main_experiments/build_veristream_evidence_index.py \
  --video-path data/ovo_bench/chunked_videos/24.mp4 \
  --output main_experiments/results/veristream_dual_role/24.index.json \
  --perception-model-path /path/to/Qwen3-VL-8B-Instruct
```

之后可使用 `main_experiments/query_veristream_dual_role.py` 执行问题级检索、局部观察和核验。
工具执行和 memory 状态转换均由本地编排器负责。
